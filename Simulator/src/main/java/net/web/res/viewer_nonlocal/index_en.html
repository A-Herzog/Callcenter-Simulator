Callcenter Simulator Viewer
Loading system

This web interface allows to load and show statistic XML files created by the Callcenter Simulator without needing to have the simulator installed.

Loading a local file into the statistic viewer

Drag & drop files here

The processing of the statistic XML file will completely be done within the browser on your computer. No data is transfered to the server.
