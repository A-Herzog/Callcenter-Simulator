Auf dieser Statistikseite werden die wichtigsten Kenngrößen für die in dem Modell enthaltenen Kundengruppen dargestellt. Es erfolgt dabei eine Unterscheidung zwischen Anrufen und Kunden. Muss ein Kunde dreimal anrufen, um letztendlich erfolgreich bedient zu werden, so stellt dies eine Erreichbarkeit von 100% auf Kundenbasis dar, aber nur eine Erreichbarkeit von 33% auf Anrufbasis.
