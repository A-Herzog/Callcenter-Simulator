Die gemessenen Mittelwerte für Erreichbarkeiten, Wartezeiten usw. beziehen sich immer auf die durchgeführten Simulationsläufe. Die konkreten Simulationsrealisierungen stellen jedoch nur einige mögliche Realisierungen dar, die sich auf Basis der Modellparameter ergeben können, so dass die tatsächlichen Mittelwerte, die sich auf Basis der Modellparameter ergeben, von den Simulationsergebnissen abweichen können.

Konfidenzintervalle geben an, mit welcher Wahrscheinlichkeit sich der tatsächliche Mittelwert in einem wie großen Bereich um den per Simulation ermittelten Mittelwert befindet. Je höher die geforderte Wahrscheinlichkeit (auch Konfidenzniveau genannt) sein soll, desto größere Intervalle ergeben sich. Höhere Anzahl an Wiederholungen des Simulationstages führen zu kleineren Konfidenzintervallen, d.h. der Ort des tatsächlichen Mittelwertes kann besser eingegrenzt werden.
