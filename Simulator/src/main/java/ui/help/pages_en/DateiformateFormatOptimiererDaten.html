File header

<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<!DOCTYPE OptimizerData SYSTEM "CallcenterSimulator_en.dtd">

The root node for an optimizer results file is <OptimizerData>.

General information

Each optimizer results file needs to contain each one of the following nodes:

  • <RunTime>: Stores the run time of the simulation in seconds.
  • <RunCount>: Stores the number of simulation rund needed in the optimization process.
  • <OptimizerSetup>: Stores the optimizer setup as described on the page Optimizer setup.

Information about the individual simulation runs

Each optimization results file contains one or more nodes of the type <SimulationRun>.

Each <SimulationRun> node has a Number attribute storing the number of the simulation run in the optimization process from which the following data are from.

Each <SimulationRun> node has exactly one <Statistic> node as a child node. The content of the <Statistic> node is described on the page Statistic data.
