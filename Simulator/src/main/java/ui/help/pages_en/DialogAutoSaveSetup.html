Simulation > Setup automatic saving

Dialog "Setup automatic saving"

If the simulation is not started by pressing F5 but by pressing Shift+F5 (or by the menu item "Run simulation and save results" from the "Simulation" menu), the statistic results will be saved directly after the simulation is finished. In this dialog you can setup which results should be saved in which format.

Save statistic results

If this option is activated, the results of each simulation will be save as xml statistic files in the specified directory. The file names will be numbered in ascending order.

Save filtered statisti results

This function allows to store some of the statistic results in a user-defined format. The results are append to the specified output file. To filter the results the first script from the fast access page of the statistic results page as well as an external script can be used.
