Waiting time tolerance calibration

The wait time tolerance of the clients is not directly measurable. In addition to using estimation methods (see load calculator dialog and the tools for waiting time tolerance estimation on the "Waiting time tolerance" page of the caller types dialog) this dialog offers to determination of the real waiting time tolerance of the clients by doing a parameter calibration .

Parameter calibration means an unknown parameter (the wait time tolerance in this case) is set in such a way that the simulated model coincides with a real model, i.e. the input parameters and the resulting accessibility of a real day has to be known to use this function. The input parameters has to be stored in form of a call center model. The accessibility from this real day has to be specified in this dialog. After clicking "Start simulation" the simulator varies the waiting time tolerances of the clients until the target accessibility values arise.
