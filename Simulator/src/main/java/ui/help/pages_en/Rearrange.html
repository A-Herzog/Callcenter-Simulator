Rearrange clients or agents

By using the function Rearrange clients or agents the fresh calls distributions can be changed in a way to match the agents working times distributions as good as possible (depeaking of the client arrivals) or the agents working times distributions can be changed to match the clients arrivals as good as possible.

By clicking the button Simulate changed model & compare results one can compare if arranging the clients or the agents leads to an improvement of the service parameters compared to the initial model.

This function is also available via the command line parameters.
