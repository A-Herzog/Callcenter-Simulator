Run network server

The Callcenter Simulator can distribute the computational work in a client server mode over multiple computers. Whether simulations should be performed on the local computer or via the network can be configurated in the Preferences dialog.

To run simulations on a remote machine in the network, on the remote computer this network sever has to be started.

The server service can offer a simulation server as well as a web server which can offer the graphical user interface of the simulator as an Java applet.

This function is also available via the command line parameters.
