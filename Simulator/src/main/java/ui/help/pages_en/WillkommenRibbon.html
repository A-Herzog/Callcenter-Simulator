Welcome to Callcenter Simulator

The Callcenter Simulator allows to model real real call center systems with all the relevant properties in the computer. This way the impact of modifications of the system can be examined easily.

First steps

  • On start up an example model is already loaded. After clicking Use simulator now to close this page you will get in the model editor from which you can view this model.
  • By clicking the Run simulation button from the toolbar you can start a simulation run of this example model at any time. When the simulation is finished    the recorded statistics results are displayed immediately.
  • Via the menu item New model with wizard of the File menu you can easily create a first, simple model of a real call center.
