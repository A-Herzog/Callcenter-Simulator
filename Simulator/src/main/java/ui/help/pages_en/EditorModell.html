Model overview

Dialog page "Model overview" of the model editor

On the dialog page "Model overview" of the call center model editor all parameters which can be calculated directly without simulating the model are displayed. Thus, e.g. the agents shift plans which are created from the number of available agents per interval and the distributions of the client arrivals and the available agent working hours are displayed throughout the day.

The function "Filter" above the tree structure at the left window half allows to select which nodes are to be displayed in the tree structure.
