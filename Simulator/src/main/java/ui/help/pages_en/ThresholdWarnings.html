Dialog "Thresholds"

In the threshold values dialog you can define values for the waiting time, the residence time, the accessibility, the service level and the workload for which you will get warnings in the statistic results if they are undershot or exceeded. You can define multiple threshold values for multiple performance indicators and you can always define two values leading each to a yellow and a red warning. You can also set up if these threshikds apply to all or to a single client or agent group and if only a specified number of intervals or the whole day is to be concerned.
