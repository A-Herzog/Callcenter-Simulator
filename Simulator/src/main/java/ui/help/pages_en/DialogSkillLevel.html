Dialog "Edit skill level"

The dialog for editing the skill levels can be opened by the dialog page Skill levels of the agents from the call center model editor. In the dialog "Edit skill level" you can set up the individual skill levels in the model.

Each skill level represents a set of skills to operate certain types of customers. Each agent group has to be assigned a skill level.

By the buttons Add skill and Remove skill one can select whilch individual skills should belong to a skill level.

Per client type these properties can be specified:

  Holding time distribution

If a client of the chosen type is served by an agent of this skill level, the actual holding time is determined by the holding time distribution defined here.

This distribution can be set up global (for the whole day). In addition, for each half-hour interval an individual distribution can be defined. If "Use global distribution for this interval" is selected, the global distribution will be used, otherwise the individual distribution for this interval will be used.

  Post processing time distribution

If a client of the chosen type is served by an agent of this skill level, the actual post processing time the agents needs before he is ready for the next client will be determined by the post processing time distribution defined here.

This distribution can be set up global (for the whole day). In addition, for each half-hour interval an individual distribution can be defined. If "Use global distribution for this interval" is selected, the global distribution will be used, otherwise the individual distribution for this interval will be used.

  Score

In each skill level for each client type which can be served by the agents of the seleczed skill level a score value can be defined. The higher this value, the more likely a waiting client will be routed to an agent of the selected skill level (if there are free agents of different skill levels).

The agent score results from the skill level score together with the call center score and the score, calculated from the idle time of the agent.
