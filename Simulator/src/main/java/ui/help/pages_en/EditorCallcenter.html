Call center and agents

Dialog page "Call center and agents" of the model editor

On the dialog page "Call center and agents" of the call center model editor the individual call centers where the calls will be processed can be defined.

Each call center can have an individual set of properties (like score values and minimum waiting times) and in particular can have an individual set of agents.

By the Add button a new call center can be created and by the Edit button existing call centers can be configured in the Edit call center dialog.

The sorting of the call centers, which can be changed by the Up and Down buttons, has no meaning for the simulation itself and is only offered to make the list more clear for the user.

The Copy function allows copy duplicate some call centers. On copying a call center the program will ask for a name for the new call center. Additionally one can set up if the number of agents should be scaled by some value on copying. All other properties are copied directly.

The Tools popup menu offers to open the "Load agents data" dialog directy. From this dialog the agents working hours of all agents from one call center can be loaded from a table at once.
