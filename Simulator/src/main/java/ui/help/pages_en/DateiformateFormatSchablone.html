File header

<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<!DOCTYPE ImportTemplate SYSTEM "CallcenterSimulator_en.dtd">

The root node for an import template is <ImportTemplate>. It contains a <ImportTemplateRecord> sub node for each information to be imported.

Records for the data to be imported

Each <ImportTemplateRecord> node has to have the following attributes:

  • Type describes what to import.
  • Parameter stores addition informationen for the data to be imported (i.e. the client type to which the data refer).
  • CellFrom and CellTo define the range of cells where the information to be imported are stored. The range has to be a part of a row or a column and is stored in the common spread sheet notation, i.e. B5 for the cell in the second column and the fifth row.
