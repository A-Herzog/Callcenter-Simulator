Load table via template

By using the Load table via template function the Callcenter Simulator can load multiple information at once from a workbook and write them into the current call center model. The workbook to be loaded can for example contain information about the client arrivals and the agents working periods in different rows or columns. On this page you can specify in which cell areas of which tables of the workbook the simulator should look for wgich information about clients, agents and skill levels. The table cells are specified in the usual spreadsheet programs form: Table!ColumnRow, where Column can be one or more letters (A, B, ..., Z, AA, AB, ...), Row is a 1-based row number and Table is the optionally quoted in single quotes name of the table in the workbook. As the table name optionally also the (1-based) number of the table in the workbook can be used. If no table name is specified, the first table in the workbook is used.

This function is also available via the command line parameters.
