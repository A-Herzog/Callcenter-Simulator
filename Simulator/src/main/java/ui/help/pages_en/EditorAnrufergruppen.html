Caller groups

Dialog page "Caller groups" of the model editor

On the dialog page "Caller groups" of the call center model editor the individual caller groups for the call centers can be defined.

Each client group can have custom properties such as the number and distribution of calls per day, waiting time tolerance of the clients etc.

By the Add button new caller groups can be created and by the Edit button existing groups can be configured in the Edit client type dialog.

The sorting of the groups, which can be changed by the Up and Down buttons, has no meaning for the simulation itself and is only offered to make the list more clear for the user.

The Copy function allows copy duplicate some agents groups. On copying a caller group the program will ask for a name for the new caller group. Additionally one can set up the size of the new group compared to the existing caller groups. All other properties are copied directly.

The Tools popup menu offers to open the "Load client data" dialog directy. From this dialog the call distribution for all client types can be loaded from a table at once.
