General data

Dialog page "General data" of the model editor"

On the dialog page "General data" parameters which does not belong to one of the other categories ("Caller groups, Call center and agents or Skill levels of the agents) can be configured.

Name of the model

The name specified in this field is used for further identification of the model only. The name will be saved in the model file in will also be shown in the statistic results so it can easily be recognized which model it is.

This name has no further meaning for the simulation ifself.

Date of the model day

In this field you can specify a date for which this simulation day should stand for. This selected date will be displayed in the statistic viewer.

The date of the model day has no further meaning for the simulation ifself.

Model description

The model description is just like the model name a free text field. It allows to keep track which of the specific characteristics of the model.

This description has no further meaning for the simulation ifself.

 Dialog "Global parameters"

By clicking the "Global parameters" button you can open the Global parameters dialog which allows to define some general parameters of the call center system and of the simulation (such as the maximum queue length and the number of repeatations to simulate).

 Dialog "Thresholds"

By clicking the "Thresholds" button you can open the Thresholds warnings dialog which allows to define threshold values for some key performance indicators. If these threshold values are exceeded or undershot, you will get a warning in the statistic results.
