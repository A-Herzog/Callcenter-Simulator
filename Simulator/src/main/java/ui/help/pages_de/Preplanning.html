Erlang-C basierte Agenten-Vorplanung

Mit Hilfe der Erlang-C basierten Vorplanung ist es möglich, die notwendige Agentenanzahl zur Erreichung bestimmter Qualitätsziele im Vorfeld einer Optimierung grob abzuschätzen, um so die Laufzeit des Optimierers zu verringern.

Bei der Vorplanung wird versucht, die Agentenanzahlen jeweils so festzulegen, dass der vorgegebene Service-Qualitätswert für alle Kundengruppen jeweils einzeln erreicht wird. Im Anschluss daran werden die Teil-Agentenzahlen kombiniert. Daher kann es beim Einsatz von Multiskill-Agenten sowie bei der sonstigen Verwendung von Modellparametern, die von den Erlang-C-Annahmen abweichen, so erheblichen Abweichungen zwischen dem so ermittelten Schätzwert und den tatsächlichen Simulationsergebnissen kommen.
