Dialog "Neues Callcenter-Model mit Assistent anlegen"

Der Dialog zum Anlege neuer Callcenter-Modell wird über den Menüpunkt "Neues Model mit Assistent anlegen" des des "Datei"-Menüs aufgerufen. In diesem Dialog kann mit wenigen Schritten ein erstes, einfaches Callcenter-Modell angelegt werden, welches dann später über den Modell-Editor sukzessiv erweitert werden kann.

Seite 1 von 4: Anrufer

Auf dieser Dialogseite kann eingestellt werde, wie viele Kunden im Mittel pro Halbstundenintervall anrufen. Im Gegensatz zu den Optionen auf den weiteren Dialogseiten, die bereits auf sinnvolle Vorgabewerte gesetzt sind, müssen hier Werte eingegeben werden, bevor aus das Callcenter-Modell erstellt werden kann.

Seite 2 von 4: Wartezeittoleranz der Kunden

Wenn die Anrufer in Ihrem Callcenter nur eine endliche Zeitspanne zu warten bereit sind (was der Normalfall ist), so können Sie hier die mittlere Wartezeittoleranz sowie die Wahrscheinlichkeit dafür, dass ein Warteabbrecher später einen weiteren Anruf tätigt, einstellen.

Seite 3 von 4: Weiterleitungen

Vielfach können Kunden durch einen Agenten nicht vollständig bedient werden und werden daher nach dem Gespräch mit einem Agenten noch einmal weitergeleitet. Auf dieser Dialogseite können Sie angebeben, bei welchem Anteil der Gespräche im Anschluss eine Weiterleitung auftritt.

Seite 4 von 4: Agenten

Abschließend können Sie noch angeben, wie lange ein Gespräch sowie die darauf folgende Nachbearbeitungszeit des Agenten dauern. Außerdem können Sie auf dieser Dialogseite indirekt angeben, wie viele Agenten in dem Callcenter arbeiten sollen. Durch die Auswahl einer Wunsch-Auslastung der Agenten legt der Simulator basierend auf den Daten zum Kundenankuftsstrom und den Bedien- und Nachbearbeitungsdauern die Anzahl an Agenten pro Halbstundenintervall fest. Da bei dieser Hochrechnung weder Schwankungen im Kundenankunftsstrom noch die Varianzen der Bedien- und Nachbearbeitungszeiten berücksichtigt werden können, wird die gewählte Wunsch-Auslastung nur ungefähr mit der Auslastung, die sich aus einer späteren Simulation ergibt, übereinstimmen.
