Modellüberblick

Dialogseite "Modellüberblick" des Modell-Editors

Auf der Dialogseite "Modellüberblick" des Callcenter-Modell-Editors können all die Kenngrößen des Callcenter-Systems angezeigt werden, die ohne vorherigen Simulationslauf direkt berechnet werden können. So können z.B. die Schichtpläne, die der Simulator aus den Agentenzahlen pro Halbstundeninervall erstellt, angezeigt werden und es können die Verteilungen der Kundenankünfte und der verfügbaren Agenten-Arbeitsstunden über den Tag angezeigt werden.

Über die Funktion "Filtern" über der Baumstruktur links kann eingestellt werden, welche Einträge in der Baumansicht angezeigt werden sollen.
