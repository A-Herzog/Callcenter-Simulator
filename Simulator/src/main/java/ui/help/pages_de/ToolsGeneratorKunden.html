Daten > Kundenankünfte aus Tabelle laden

Dialog "Kundendaten laden"

Über den Dialog "Kundendaten laden" können die Ankunftsverteilungen aller Kundentypen eines Callcenters auf einmal aus einer Tabelle geladen werden.

Zunächst muss in dem Feld Tabellendatei der Name der Tabellendatei, aus der die Daten geladen werden sollen, angegeben werden. Danach muss über die dann erscheinenden Auswahlfelder unterhalb der Überschrift Kundentyp gewählt werden, welche Tabellenspalten welchem Kundentypen zugeordnet werden sollen. Der Callcenter Simulator versucht dabei an Hand der Spaltenüberschriften in der Tabellendatei bereits eine sinnvolle Voreinstellung vorzunehmen.

Aufbau der Tabellendatei

Damit die Kundenankunftsverteilungen aller Kundentypen aus ein Tabellendatei geladen werden können, muss diese einen bestimmten Aufbau besitzen. Die Kundenankunftsverteilung für einen Kundentyp besteht aus 48 Zahlen, die angeben, wie viele Kunden in dem jeweiligen Halbstundenintervall eintreffen sollen. Diese 48 Zahlenwerte müssen in der Tabelle in 48 unmittelbar aufeinander folgenden Zeilen einer Spalte stehen. In jeder Spalte können die Daten zu einem Kundentyp stehen. Sollen mehrere Kundentypen auf einmal geladen werden, so müssen weitere Spalten in der Tabelle verwendet werden. Die Spalten müssen nicht unmittelbar benachbart sein, die Reihe der Zahlenwerte muss jedoch jeweils in derselben Zeile beginnen. Alle nichtleeren Zeilen oberhalb der ersten Zahlen-Zeile werden als Spaltenüberschrift interpretiert. Die jeweiligen Spaltenüberschriften werden im "Kundendaten laden"-Dialog unter der Überschrift Tabellenspalte angezeigt. Steht in einer Spalte mit 48 Zahlenwerten keine Zeichenkette in einer der darüberliegenden Zeilen, so wird die Spalten-Nummer (z.B. "Spalte A", "Spalte B" usw.) angezeigt.
