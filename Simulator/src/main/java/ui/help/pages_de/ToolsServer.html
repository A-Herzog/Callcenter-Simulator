Netzwerk-Server starten

Der Callcenter Simulator kann im Client-Server-Modus grafische Oberfläche und die zugehörigen Rechenarbeiten auf verschiedene Computer aufteilen. Ob Simulationen auf dem lokalen Rechner oder auf einem Rechner im Netz durchgeführt werden sollen, lässt sich im Einstellungen-Dialog konfigurieren.

Soll auf einem entfernen Rechner im Netzwerk simuliert werden, so muss auf diesem der über die entsprechende Funktion im Tools-Menü verfügbare Netzwerk-Server gestartet sein.

Der Serverdienst kann dabei sowohl einen Rechenserver als auch einen Webserver, über den die grafische Oberfläche des Simulators als Java-Applet zur Verfügung gestellt werden kann, zur Verfügung stellen.

Diese Funktion kann auch über Kommandozeilenparameter verwendet werden.
