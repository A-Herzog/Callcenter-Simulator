Anrufer oder Agenten verlagern

Der Callcenter Simulator kann über die Funktion Anrufer oder Agenten verlagern entweder die Erstanrufer-Verteilung so anpassen, dass diese gut zu der Verteilung der Arbeitszeiten der Agenten passt (Depeaking der Kunden) oder aber die Agenten-Arbeitszeitverteilung so anpassen, dass diese möglichst gut zu der Verteilung der Kundenankünfte passt.

Über die Funktion Angepasstes Modell simulieren & Ergebnisse vergleichen kann verglichen werden, ob die Umverteilung der Kunden bzw. der Agenten eine Verbesserung gegenüber dem Ausgangsmodell zur Folge hat.

Diese Funktion kann auch über Kommandozeilenparameter verwendet werden.
