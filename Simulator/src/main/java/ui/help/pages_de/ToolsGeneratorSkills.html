Daten > Bedien- und Nachbearbeitungszeiten aus Tabelle laden

Dialog "Skilldaten laden"

Über den Dialog "Skilldaten laden" können die Bedien- und Nachbearbeitungszeitverteilungen, die die Agenten bestimmter Skill-Level für die Bedienung bestimmter Kundentypen benötigen auf einmal aus einer Tabelle geladen werden.

Zunächst muss in dem Feld Tabellendatei der Name der Tabellendatei, aus der die Daten geladen werden sollen, angegeben werden. Danach muss über die dann erscheinenden Auswahlfelder unterhalb der Überschrift Verteilung gewählt werden, welche Tabellenspalten welcher Bedien- oder Nachbearbeitungszeitverteilung zugeordnet werden sollen.

Aufbau der Tabellendatei

Damit die Bedien- und Nachbearbeitungsverteilungen für mehrere Kundentypen und Skill-Level aus ein Tabellendatei geladen werden können, muss diese einen bestimmten Aufbau besitzen. Die Bedien- und Nachbearbeitungsverteilungen für einen Kundentyp besteht aus maximal 3600 Zahlen, die eine sekundenbasierende Zähldichte dafür angeben, wie lange die jeweiligen Bearbeitungsschritte für einen bestimmten Kundentyp durchgeführt von einem Agenten eines bestimmten Skill-Levels dauern. Diese maximal 3600 Zahlenwerte müssen in der Tabelle in unmittelbar aufeinander folgenden Zeilen einer Spalte stehen. In jeder Spalte können die Daten für eine Verteilung stehen. Sollen mehrere Datensätze auf einmal geladen werden, so müssen weitere Spalten in der Tabelle verwendet werden. Die Spalten müssen nicht unmittelbar benachbart sein, die Reihe der Zahlenwerte muss jedoch jeweils in derselben Zeile beginnen. Alle nichtleeren Zeilen oberhalb der ersten Zahlen-Zeile werden als Spaltenüberschrift interpretiert. Die jeweiligen Spaltenüberschriften werden im "Skilldaten laden"-Dialog unter der Überschrift Tabellenspalte angezeigt. Steht in einer Spalte mit den Zahlenwerten keine Zeichenkette in einer der darüberliegenden Zeilen, so wird die Spalten-Nummer (z.B. "Spalte A", "Spalte B" usw.) angezeigt.
