Dialog "Agentengruppe bearbeiten"

Der Dialog zum Bearbeiten von Agentengruppen wird über die Dialogseite "Agenten" des Callcenter bearbeiten-Dialogs aufgerufen. In dem Dialog "Agentengruppe bearbeiten" können Einstellungen zu einer einzelnen Agentengruppe vorgenommen werden.

Agentengruppe aktiv

Über dieses Auswahlfeld kann eingestellt werden, ob diese Agentengruppe bei der Simulation berücksichtigt werden soll. Durch das gezielte Aktivieren und Deaktivieren von Agentengruppen können schnell verschiedene Szenarien ausprobiert werden, ohne dass dafür verschiedene Modelle angelegt werden müssen.

Dialogseite    "Arbeitszeiten"

Während der Simulation verwendet der Simulator stets konkrete Agenten mit vorgegebenen Arbeitszeiten. Da aber vielfach die Arbeitszeiten der einzelnen Agenten nicht bekannt sind und somit nicht als Ausgangsdaten zur Verfügung stehen, kann auch eine Verteilung der Agenten über den Tag angegeben werden, so der der Simulator dann vor dem Beginn des Simulationsprozesses konkrete Agenten erstellt. (Siehe dazu auch die Optionen zur Schichtplanung auf der Dialogseite Allgemeine Daten des Callcenter-Modell-Editors.) Noch einen Schritt weiter geht die Option "Verteilung über den Tag gemäß Kundenankünften", bei deren Verwendung noch nicht einmal eine Verteilung der Agenten über den Tag explizit angegeben werden muss, sondern diese vom Simulator so angelegt wird, dass sie der Ankunftsverteilung eines oder mehrerer Kundentypen entspricht.

Agenten mit festen Arbeitszeiten

Ist die Option "Agenten mit festen Arbeitszeiten" gewählt, so kann in dem Dialog angegeben werden, aus vie vielen Agenten die Agentengruppe bestehen soll und von wann bis wann die Dienstzeit der Agenten reichen soll. Außerdem kann über das Auswahlfeld Arbeitszeitende ignorieren und beliebig weiterarbeiten eingestellt werden, dass die Agenten ab dem angegebenen Arbeitszeitbeginn bis zum Ende des Simulationstages arbeiten.

Vorgegebene Verteilung über den Tag

Ist die Option "Vorgegebene Verteilung über den Tag" gewählt, so kann angegeben werde, wie viele Agenten in der jeweiligen Gruppe pro Intervall aktiv sein sollen. Vor Beginn der eigentlichen Simulation werden dann konkrete Agenten mit festen Arbeitszeiten aus diesen Daten generiert. Um dem realen Verhalten eines Callcenters möglichst nahe zu kommen, werden keine Agenten mit Halbstundenschichten erzeugt, sondern es wird versucht die auf der Dialogseite Allgemeine Daten des Callcenter-Modell-Editors bzw. die auf dieser Seite einstellbare Schichtlänge einzuhalten. Über die Schaltfläche Schichtplan anzeigen kann der aus den Daten erzeugte Schichtplan sofort eingesehen werden.

Desweiteren kann eingestellt werden, ob die Agenten, die regulär bis 24 Uhr arbeiten weiter arbeiten sollen, bis keine Kunden mehr im System sind. Wird lediglich ein einzelner Tag simuliert, so sollte diese Option aktiviert werden, um zu vermeiden, dass im letzten Intervall auf einmal Abbrecher auftreten, die es im realen System (in dem sich der nächste Tag anschließt) nicht gibt. Wird ein über mehrere Tage verknüpftes System simuliert, so kann diese Option deaktiviert werden, da in diesem Fall die Kunden auf den nächsten Tag übertragen werden können.

Verteilung über den Tag gemäß Kundenankünften

Ist die Option "Verteilung über den Tag gemäß Kundenankünften" gewählt, so wird die verfügbare Agenten-Arbeitsleistung, die über das Eingabefeld Verfügbare Agenten-Halbstunden eingestellt werden kann, gemäß der Verteilung der Erstanrufer bestimmter Kundentypen über den Tag verteilt. Wie stark welcher Kundentyp in die Berechnung der Verteilung der Agenten einfließen soll, kann über die Rate-Felder eingestellt werden. Je größer die Rate für einen Kundentyp ist, desto stärker fließt er in die Berechnung der Agentenverteilung mit ein. Um dem realen Verhalten eines Callcenters möglichst nahe zu kommen, werden keine Agenten mit Halbstundenschichten erzeugt, sondern es wird versucht die auf der Dialogseite Allgemeine Daten des Callcenter-Modell-Editors bzw. die auf dieser Seite einstellbare Schichtlänge einzuhalten. Über die Schaltfläche Schichtplan anzeigen kann der aus den Daten auf Halbstundenbasis erzeugte Schichtplan sofort eingesehen werden.

In dem Modus "Verteilung über den Tag gemäß Kundenankünften" kann über die Funktion Anzahl gemäß rechnerischer Auslastung, die über die Tools-Schaltfläche aufgerufen werden kann, automatisch berechnet werden, wie viele Agenten notwendig sind, um einen gewissen Anteil der Kunden bedienen zu können. Da diese Rechnung statisch erfolgt (mittlere Gesprächs- und Nachbearbeitungsdauer multipliziert mit der Anzahl der Anrufer) und weder Schwankungen beim Anrufeingang noch Schwankungen in den Bediendauern berücksichtigt, liefert diese Funktion lediglich eine untere Abschätzng für die benötigte Anzahl an Agenten.

Produktivität

In diesem Dialog können Sie einstellen, wie viel der angegebenen Agentenarbeitsleitung tatsächlich verfügbar sein soll. Die Angaben beziehen sich auf die aktuelle Agentengruppe und stehen nur zur Verfügung, wenn die Gruppe nicht über feste Arbeitszeiten definiert ist. Es kann angegeben werden, dass die Agenten stets wie angegeben verfügbar sind, dass sie nur für einen bestimmten, festen Anteil der angegebenen Leistung verfügbar sind oder aber dass die Produktivität in Halbstundenintervallen variiert. Des Weiteren kann eingestellt werden, dass die globalen Werte verwendet werden sollen und keine besonderen Parameter für dieses Agentengruppe gelten sollen.

Krankheitszuschlag

In diesem Dialog können Sie einstellen, wie viel Agenten prozentual auf die vorgegebenen Agentenverteilungen aufgeschlagen werden sollen, um krankheitsbedingte Ausfälle zu kompensieren. Die daraus resultierende erhöhte Agentenverteilung wird nicht in der Simulation verwendet, sondern nur zusätzlich in der Statistik ausgewiesen. Die Angaben beziehen sich auf die aktuelle Agentengruppe und stehen nur zur Verfügung, wenn die Gruppe nicht über feste Arbeitszeiten definiert ist. Es kann angegeben werden, dass kein Aufschlag erfolgen soll, dass diese prozentual in allen Intervallen identisch sein soll aber dass der Zuschlag in Halbstundenintervallen variiert.

Dialogseite    "Skill-Level"

Jeder Agentengruppe muss einem Skill-Level zugeordnet werden, um festzulegen, welche Kundentypen Agenten aus dieser Gruppe bedienen können.

Über die Schaltfläche Skill-Level bearbeiten kann unmittelbar der Dialog zur Bearbeitung eines Skill-Levels geöffnet werden, so dass der gewählte Skill-Level ohne Umwege bearbeitet werden kann.

Dialogseite    "Kosten"

Jeder Agentengruppe kann ein Stundenlohn zugewiesen werden sowie Kosten, die pro Anruf und pro Gesprächs- und Nachbearbeitungsminute entfallen. Die Kosten für Gespräche, sowie Gesprächs- und Nachbearbeitungszeit können dabei pro Kundentyp angegeben werden.
