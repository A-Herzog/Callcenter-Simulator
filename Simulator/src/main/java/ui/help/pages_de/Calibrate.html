Wartezeittoleranzkalibrierung

Die Wartetzeittoleranz der Kunden ist systembedingt nicht messbar. Neben der Verwendung von Schätzverfahren (siehe Auslastungsrechner und Funktion zur Wartezeittoleranz-Schätzung auf der Dialogseite "Wartezeittoleranz" des Kundentypen-Dialogs) ist die übliche Methode zur Ermittlung der realen Wartezeittoleranz der Kunden die Parameterkalibrierung, die über diese Programmfunktion angeboten wird.

Bei der Parameterkalibrierung wird ein unbekannter Parameter (in diesem Fall die Wartezeittoleranz) so eingestellt, dass das simulierte Modell mit einem realen Modell übereinstimmt, d.h. es müssen von einem realen Tag sowohl die Eingangsgrößen als auch die sich daraus ergebenden Erreichbarkeiten bekannt sein. Die Eingangsgrößen müssen in Form eines Callcenter Modells im Simulator hinterlegt werden. Die Erreichbarkeiten, die sich bei diesen Eingangsgrößen in der Realität ergeben haben, werden in dieser Programmfunktion eingestellt. Nach einem Klick auf "Simulation starten" variiert der Simulator die Wartezeittoleranzen der Kunden so lange, bis sich die Ziel-Erreichbarkeitswerte ergeben.
