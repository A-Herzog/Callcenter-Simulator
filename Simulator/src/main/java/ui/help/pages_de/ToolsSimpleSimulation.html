Simulation eines Einfach-Modells

Diese Funktion ermöglicht es, ein einfaches Modell bestehend aus nur einer Kunden- und einer Agentengruppe in Echtzeit zu simulieren. Sobald Sie einen der Eingabeparameter auf der linken Seite ändern, wird im Hintergrund eine Simulation durchgeführt und auf der rechten Seite des Dialogs werden unmittelbar die Ergebnisse angezeigt. Auf diese Weise kann sehr schnell und einfach untersucht werden, welche Auswirkungen Änderungen an dem Modell haben.
