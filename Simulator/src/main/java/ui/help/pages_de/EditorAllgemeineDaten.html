Allgemeine Daten

Dialogseite "Allgemeine Daten" des Modell-Editors"

Auf der Dialogseite "Allgemeine Daten" des Callcener-Modell-Editors können einige generelle Einstellungen vorgenommen werden, die nicht in einer der anderen Kategorien ("Anrufergruppen, Callcenter und Agenten oder Skill-Level) fallen.

Name des Modell

Der in diesem Feld angegebene Name dient nur zur späteren Identifikation des Modells. Er wird in der Modell-Datei gespeichert und nach der Simulation auch in die Statistik-Ergebnisse übernommen, so dass leicht wiedererkannt werden kann, um welches Modell es sich handelt.

Der Name hat für die Simulation selbst keien weitere Bedeutung.

Datum des Modelltages

In diesem Feld kann ein Datum ausgewählt werden, welches angibt, welchen realen Tag das Simulationsmodell repräsentieren soll. Diese Angabe wird in die Statistik übernommen.

Die Datumsangabe hat für die Simulation selbst keien weitere Bedeutung.

Beschreibung

Bei der Modell-Beschreibung handelt es sich genau so wie bei dem Namen um ein Freitextfeld. Durch die Möglichkeit, eine Modell-Beschreibung als Text direkt in der Modell-Datei zu speichern, ist es später einfacher, den Überblick darüber zu behalten, welche besonderen Eigenschaften ein bestimmtes Modell jeweils auszeichnen.

Diese Beschreibung hat für die Simulation selbst keine weitere Bedeutung.

 Dialog "Globale Parameter"

Wenn Sie auf die Schaltfläche "Globale Parameter" klicken, öffent sich der Globale Parameter Dialog, in dem Sie einige generelle Parameter des Callcenter-Systems und der Simulation festlegen können (wie z.B. die maximale Warteschlangenlänge und die Anzahl an Wiederholungen des Simulationstages).

 Dialog "Schwellenwerte"

Wenn Sie auf die Schaltfläche "Schwellenwerte" klicken, öffent sich der Schwellenwert-Warnungen Dialog, in dem Sie Schwellenwerte für einzelne Kenngrößen einstellen können. Wenn diese Schwellenwerte über- oder unterschritten werden, erhalten Sie in den Statistikergebnissen eine entsprechende Warnung.
