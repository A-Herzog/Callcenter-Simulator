Simulationsergebnisse

Statistik-Betrachter

Der Statistik-Betrachter wird direkt nach dem Ende eines Simulationslaufes angezeigt. Er stellt alle im Simulationsprozess erhobenen Daten dar.

Auf der linken Seite des Fensters ist eine Baumstruktur zu sehen, in der alle verfügbaren Themen aufgelistet wird. Wird eines dieser Themen angeklickt, so erscheinen die Daten zu diesem Thema auf der rechten Fensterhälfte. Über die Funktion "Filtern" über der Baumstruktur kann eingestellt werden, welche Einträge in der Baumansicht angezeigt werden sollen.

Statisik-Daten können auch gespeichert und später wieder geladen werden. Dies ist insbesondere dann nützlich, wenn die eigentliche Simulation vorab auf einem anderen Rechner z.B. im Kommandozeilenmodus erfolgt ist.

Über die Symbollestenschaltfläche Modell zu diesen Ergebnissen lässt sich das Callcenter-Modell, welches diesen Statistik-Daten zugrunde liegt anzeigen und auch wieder in den Editor laden.
