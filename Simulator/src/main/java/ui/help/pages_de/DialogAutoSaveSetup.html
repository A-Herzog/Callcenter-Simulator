Simulation > Automatische Speicherung konfigurieren

Dialog "Automatische Speicherung konfigurieren"

Wenn Sie die Simulation nicht per F5 sondern per Umschalt+F5 starten (bzw. über die Funktion "Simulation starten und Ergebnisse speichern" aus dem "Simulation"-Menü), werden die Ergebnisse der Simulation direkt nach deren Abschluss gespeichert. In diesem Dialog können Sie einstellen, welche Ergebnisse in welcher Form und wo gespeichert werden solln.

Statistik-Ergebnisse speichern

Ist diese Option aktiviert, so werden die Ergebnisse jeder Simulation als xml-Datei in dem angegebenen Verzeichnis gespeichert. Die Dateinamen werden dabei aufsteigend durchnummeriert.

Gefilterte Statistik-Ergebnisse speichern

Über diese Funktion können bestimmte Ergebnisse in einer vom Benutzer gewählten Form gespeichert werden. Die Ergebnisse werden dabei an die angegebene Ausgabedatei angehängt. Zur Filterung der Ergebnisse kann dabei sowohl das erste unter Schnellzugriff auf der Statistikseite angegebene Skript als auch ein externes Script verwendet werden.
