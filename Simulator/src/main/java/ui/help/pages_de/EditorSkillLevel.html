Skill-Level

Dialogseite "Skill-Level" des Modell-Editors

Auf der Dialogseite "Skill-Level" des Callcener-Modell-Editors können die in dem Callcenter-Verbund auftretenden Agenten-Skill-Levels definiert werden. Agenten eines jeden Skill-Level können eine bestimmte Teilmenge der auftretenden Kundentypen bedienen. Auch ist in dem jeweiligen Skill-Level festgelegt, gemäß welchen Bedienzeit- und Nachbearbeitungszeitverteilungen die Agenten die Kunden bedienen.

Wird in einem Callcenter eine Agentengruppe angelegt, so muss ihr eines der hier definierten Skill-Level zugewiesen werden.

Über die Schaltflächen Hinzufügen können neue Skill-Level angelegt werden und über die Schalltfläche Bearbeiten bestehene Skill-Level im Skill-Level bearbeiten-Dialog konfiguriert werden.

Die Reihenfolge der Skill-Level, die über die Nach oben und Nach unten Schaltflächen eingestellt werden kann, hat keinen Einfluss auf die Simulation und dient nur dazu, die Liste für den Benutzer möglichst übersichtlich gestaltbar zu machen.

Die Kopieren-Funktion ermöglicht es, einzelne Skill-Level zu kopieren. Beim Kopieren muss ein neuer Name für den neuen Skill-Level angegeben werden. Alle weiteren Eigenschaften werden direkt kopiert.

Das Tools-Popupmenü bietet die Möglichkeit, den "Skillladen"-Dialog direkt aufzurufen. Über diesen Dialog können die Bedien- und Nachbearbeitungszeiten mehrerer Skill-Levels auf einmal aus einer Tabelle geladen werden.
