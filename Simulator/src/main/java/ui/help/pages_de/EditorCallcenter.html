Callcenter und Agenten

Dialogseite "Callcenter und Agenten" des Modell-Editors

Auf der Dialogseite "Callcenter und Agenten" des Callcener-Modell-Editors können die einzelnen Callcentern, in denen die Gespräche abgewickelt werden sollen, definiert werde.

Jedes Callcenter kann über individuell Eigenschaften (wie z.B. Score-Werte und Mindestwartezeiten) verfügen und vor allem einen eigenen Satz an Agenten beherrbergen.

Über die Schaltflächen Hinzufügen können neue Callcenter angelegt werden und über die Schalltfläche Bearbeiten bestehene Callcenter im Callcenter bearbeiten-Dialog konfiguriert werden.

Die Reihenfolge der Callcenter, die über die Nach oben und Nach unten Schaltflächen eingestellt werden kann, hat keinen Einfluss auf die Simulation und dient nur dazu, die Liste für den Benutzer möglichst übersichtlich gestaltbar zu machen.

Die Kopieren-Funktion ermöglicht es, einzelne Callcenter zu kopieren. Beim Kopieren muss ein neuer Name für das neue Callcenter angegeben werden und es kann eingestellt werden, ob die Agentenzahlen beim Kopieren mit einem bestimmten Wert skaliert werden sollen. Alle weiteren Eigenschaften werden direkt kopiert.

Das Tools-Popupmenü bietet die Möglichkeit, den "Agentendaten laden"-Dialog direkt aufzurufen. Über diesen Dialog können die Agenten-Arbeitszeiten aller Agentengruppen eines Callcenters auf einmal aus einer Tabelle geladen werden.
