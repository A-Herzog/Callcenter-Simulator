Datei > Neues Modell

Dialog "Neues Callcenter-Model anlegen"

Der Dialog zum Anlege neuer Callcenter-Modell wird über den Menüpunkt "Neues Model" des "Datei"-Menüs aufgerufen. In diesem Dialog kann gewählt werden, ob ein leeres Modell erstellt werden soll oder ob das Modell auf einer Vorlage basieren soll.
