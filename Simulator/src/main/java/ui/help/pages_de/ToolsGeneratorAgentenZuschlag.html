Dialog "Krankheitsbedingten Aufschlag auf Agentenanzahl laden"

Über den Dialog "Krankheitsbedingten Aufschlag auf Agentenanzahl laden" kann der planerische Aufschlag auf die Agentenverteilung zum Ausgleich von krankheitsbedingten ausfällen für alle Agentengruppen eines Callcenters aus einer Tabelle geladen werden. Die aus der Modell-Agentenverteilung und diesem prozentualen Aufschlag resultierende Verteilung wird nicht für die Simulation verwendet, sondern lediglich in der Statistik mit ausgewiesen.

Zunächst muss in dem Feld Tabellendatei der Name der Tabellendatei, aus der die Daten geladen werden sollen, angegeben werden. Danach muss über das Auswahlfeld unter Callcenter festgelegt werden, für welches Callcenter der Aufschlag verwendet werden soll und es muss über die dann erscheinenden Auswahlfelder unterhalb der Überschrift Agentengruppe gewählt werden, welche Tabellenspalten welcher Agentengruppe zugeordnet werden sollen. Der Callcenter Simulator versucht dabei an Hand der Spaltenüberschriften in der Tabellendatei bereits eine sinnvolle Voreinstellung vorzunehmen.

Aufbau der Tabellendatei

Damit der krankheitsbedingte Zuschlag auf alle Agentengruppen eines Callcenters aus ein Tabellendatei geladen werden kann, muss die Tabelle einen bestimmten Aufbau besitzen. Der Zuschlag für einer Agentengruppe besteht aus 48 Zahlen, die angeben, wie hoch diese in dem jeweiligen Halbstundenintervall sein sollen, wobei "100%" bedeutet, dass kein Zuschlag erfolgt. Diese 48 Zahlenwerte müssen in der Tabelle in 48 unmittelbar aufeinander folgenden Zeilen einer Spalte stehen. In jeder Spalte können die Daten zu einer Agentengruppe stehen. Sollen mehrere Agentengruppen auf einmal geladen werden, so müssen weitere Spalten in der Tabelle verwendet werden. Die Spalten müssen nicht unmittelbar benachbart sein, die Reihe der Zahlenwerte muss jedoch jeweils in derselben Zeile beginnen. Alle nichtleeren Zeilen oberhalb der ersten Zahlen-Zeile werden als Spaltenüberschrift interpretiert. Die jeweiligen Spaltenüberschriften werden im "Agentenproduktivität laden"-Dialog unter der Überschrift Tabellenspalte angezeigt. Steht in einer Spalte mit 48 Zahlenwerten keine Zeichenkette in einer der darüberliegenden Zeilen, so wird die Spalten-Nummer (z.B. "Spalte A", "Spalte B" usw.) angezeigt. Bestehen die Überschriften aus den Namen der Kundentypen, die die jeweilige Agentengruppen bedienen können (pro Zeile eine), so versucht der Callcenter Simulator automatisch vom Skill-Level her passende Agentengruppen in den Auswahlfeldern unter Agentengruppe vorzuselektieren.
