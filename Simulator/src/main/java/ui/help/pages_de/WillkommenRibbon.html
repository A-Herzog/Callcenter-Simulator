Willkommen im Callcenter Simulator

Der Callcenter Simulator ermöglicht es, reale Callcenter-Systeme mit allen relevanten Eigenschaften im Computer abzubilden. Auf diese Weise kann leicht virtuell untersucht werden, welche Auswirkungen Veränderungen an dem System hätten.

Erste Schritte

  • Beim Starten des Simulators wird bereits ein Beispiel-Modell geladen. Nach dem Sie auf Simulator jetzt verwenden geklickt haben, um diese Informationsseite zu schließen, gelangen Sie diekt in den Modell-Editor, in dem Sie sich dieses Modell ansehen können.
  • Über die Schaltfläche Simulation starten aus der Symbolleiste können Sie sofort einen Simulationslauf mit diesem Beispiel-Modell starten. Nach dem Ende der Simulation werden sofort die aufgezeichneten Statistik-Ergebnisse angezeigt.
  • Über den Menüpunkt Modell mit Assistent anlegen des Datei-Menüs können Sie mit wenigen Mausklicks ein erstes, einfaches Modell Ihres realen Callcenters anlegen.

Häufig gestellte Fragen

  • Wie funktioniert der Simulator? Was läst sich mit dem Simulator untersuchen? → siehe Ereignisorientierte Simulation
  • Wie sieht das simulierte Callcenter-Modell aus bzw. gibt es eine Übersicht über die abgebildeten Eigenschaften? → siehe Simulationsmodell
