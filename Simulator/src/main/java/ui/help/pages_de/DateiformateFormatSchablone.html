Das Basis-Element für Import-Schablonen heißt <Schablone>, welches für jedes zu importierende Datenelement ein <SchabloneEintrag>-Unterelement besitzen muss.

Einträge für die zu importierenden Daten

Jedes <SchabloneEintrag>-Element muss genau die vier folgenden Attribute besitzen:

  • Typ gibt an, was importiert werden soll.
  • Parameter gibt zusätzliche Informationen zu den zu importierenden Daten an (z.B. auf welchen Kundentyp sich die Daten beziehen sollen).
  • ZelleVon und ZelleBis geben den Datenbereich an, aus dem die Informationen importiert werden sollen. Der Bereich muss entweder eine Teil einer Zeile oder einer Spalte sein und wird in Tabellenkalkulationsprogramm-Notation angegeben, also z.B. B5 für die Zelle in der zweiten Spalte und der 5. Zeile.
