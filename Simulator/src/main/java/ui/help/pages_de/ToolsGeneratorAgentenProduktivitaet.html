Daten > Agentenproduktivität aus Tabelle laden

Dialog "Agentenproduktivität laden"

Über den Dialog "Agentenproduktivität laden" kann die Produktivität für alle Agentengruppen eines Callcenters aus einer Tabelle geladen werden.

Zunächst muss in dem Feld Tabellendatei der Name der Tabellendatei, aus der die Daten geladen werden sollen, angegeben werden. Danach muss über das Auswahlfeld unter Callcenter festgelegt werden, für welches Callcenter die Agentenproduktivität verwendet werden soll und es muss über die dann erscheinenden Auswahlfelder unterhalb der Überschrift Agentengruppe gewählt werden, welche Tabellenspalten welcher Agentengruppe zugeordnet werden sollen. Der Callcenter Simulator versucht dabei an Hand der Spaltenüberschriften in der Tabellendatei bereits eine sinnvolle Voreinstellung vorzunehmen.

Aufbau der Tabellendatei

Damit die Agentenproduktivitäten aller Agentengruppen eines Callcenters aus ein Tabellendatei geladen werden können, muss diese einen bestimmten Aufbau besitzen. Die Agentenproduktivität für einer Agentengruppe besteht aus 48 Zahlen, die angeben, wie hoch diese in dem jeweiligen Halbstundenintervall sein sollen. Diese 48 Zahlenwerte müssen in der Tabelle in 48 unmittelbar aufeinander folgenden Zeilen einer Spalte stehen. In jeder Spalte können die Daten zu einer Agentengruppe stehen. Sollen mehrere Agentengruppen auf einmal geladen werden, so müssen weitere Spalten in der Tabelle verwendet werden. Die Spalten müssen nicht unmittelbar benachbart sein, die Reihe der Zahlenwerte muss jedoch jeweils in derselben Zeile beginnen. Alle nichtleeren Zeilen oberhalb der ersten Zahlen-Zeile werden als Spaltenüberschrift interpretiert. Die jeweiligen Spaltenüberschriften werden im "Agentenproduktivität laden"-Dialog unter der Überschrift Tabellenspalte angezeigt. Steht in einer Spalte mit 48 Zahlenwerten keine Zeichenkette in einer der darüberliegenden Zeilen, so wird die Spalten-Nummer (z.B. "Spalte A", "Spalte B" usw.) angezeigt. Bestehen die Überschriften aus den Namen der Kundentypen, die die jeweilige Agentengruppen bedienen können (pro Zeile eine), so versucht der Callcenter Simulator automatisch vom Skill-Level her passende Agentengruppen in den Auswahlfeldern unter Agentengruppe vorzuselektieren.
