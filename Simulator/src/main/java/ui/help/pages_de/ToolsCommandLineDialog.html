Kommandozeilenbefehl ausführen

In diesem Dialog werden alle Befehle, die als Kommandozeilenparameter beim Aufruf des Callcenter Simulators genutzt werden können, aufgelistet. Zusätzlich wird zu allen Befehlen die Beschreibung der Funktionsweise angezeigt. Die Befehle können auch direkt innerhalb dieses Dialogs ausgeführt werden.
