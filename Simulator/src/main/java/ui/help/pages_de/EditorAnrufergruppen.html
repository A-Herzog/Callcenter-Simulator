Anrufergruppen

Dialogseite "Anrufergruppen" des Modell-Editors

Auf der Dialogseite "Anrufergruppen" des Callcener-Modell-Editors können die einzelnen Kundengruppen, die bei dem Callcenter anrufern, definiert werde.

Jede Kundengruppe kann individuelle Eigenschaften wie Anzahl und Verteilung der Anrufe pro Tag, Wartebereitschaft der Kunden usw. besitzen.

Über die Schaltflächen Hinzufügen können neue Anrufergruppen angelegt werden und über die Schaltfläche Bearbeiten bestehende Gruppen im Kundentyp bearbeiten-Dialog konfiguriert werden.

Die Reihenfolge der Gruppen, die über die Nach oben und Nach unten Schaltflächen eingestellt werden kann, hat keinen Einfluss auf die Simulation und dient nur dazu, die Liste für den Benutzer möglichst übersichtlich gestaltbar zu machen.

Die Kopieren-Funktion ermöglicht es, einzelne Anrufergruppen zu kopieren. Beim Kopieren muss ein neuer Name für die neue Anrufergruppe angegeben werden und es kann eingestellt werden, wie groß die neue Gruppe im Vergleich zu der bestehenden Gruppe sein soll. Alle weiteren Eigenschaften werden direkt kopiert.

Das Tools-Popupmenü bietet die Möglichkeit, den "Kundendaten laden"-Dialog direkt aufzurufen. Über diesen Dialog können die Kundenankunftsverteilungen aller Kundentypen auf einmal aus einer Tabelle geladen werden.
