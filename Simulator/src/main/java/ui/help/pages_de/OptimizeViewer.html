Optimierer Ergebnisse

Ein Optimierungslauf besteht aus einer Reihe von Simulationsläufen mit jeweils leicht verändertem Callcenter-Modell. Als Ergebnis eines Optimierungsprozesses sind üblicherweise die Ergebnisse eines einzelnen Laufes weniger von Interesse; viel mehr interessiert man sich für die Veränderungen wichtiger Kenngrößen von Simulationslauf zu Simulationslauf. Genau diese Informationen liefert der Ergebnisbetrachter des Optimierers.

Auf der linken Seite des Fensters ist eine Baumstruktur zu sehen, in der alle verfügbaren Themen aufgelistet wird. Wird eines dieser Themen angeklickt, so erscheinen die Daten zu diesem Thema auf der rechten Fensterhälfte.

Optimierungsergebnisse-Daten können auch gespeichert und später über den Menüpunkt Optimierer-Ergebnis anzeigen des Simulation-Menüs wieder geladen werden. Außerdem können die Ergebnisse eines einzelnen Simulationslaufs über die Schaltfläche Ergebnise für einzelnen Simulationslauf angezeigt werden und auch die zugrunde liegenden Modelle einzelner Simulationsläufe in den Modell-Editor geladen werden.
