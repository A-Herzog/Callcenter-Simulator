Verteilung anpassen

Diese Funktion ermöglicht es zu ermitteln, welche der im Callcenter Simulator zur Verfügung stehenden Wahrscheinlichkeitsverteilungen am besten zu einer vorgegebenen Menge an Messwerten passt.

Zur Nutzung dieser Funktion müssen lediglich eine Reihe von Messwerten aus der Zwischenablage oder aus einer Datei geladen werden. Es können direkt erfasste Messwerte (eine Zeile oder Spalte) oder aber aufsummierte Messwerte (zwei Zeilen oder Spalten; in der ersten die Werte und in der zweiten deren Häufigkeiten) verarbeitet werden. Es wird dann daraus eine empirische Verteilung erstellt und auf Basis dieser Verteilung wird dann der mittlere quadratische Abstand zu den im Callcenter Simulator verfügbaren analytischen Wahrscheinlichkeitsverteilungen (mit jeweils basierend auf den Messwerten sinnvoll gewählten Parametern) bestimmt.
