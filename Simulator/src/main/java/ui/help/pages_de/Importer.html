Tabelle gemäß Schablone laden

Der Callcenter Simulator kann über die Funktion Tabelle gemäß Schablone laden mehrere Informationen aus einer Arbeitsmappe laden und in das aktuelle Modell schreiben. In der Ausgangsarbeitsmappe können sich z.B. Informationen über die Kundenankünfte und die Agentenarbeitszeiten in verschiedenen Zeilen oder Spalten befinden. Auf dieser Seite des Callcenter Simulators können Sie einstellen, in welchem Bereich der Simulator nach welchen Informationen suchen soll und sie welchen Kunden, Agenten oder Skill-Level Bereichen zuordnen soll. Die Tabellenfelder werden dabei in der in Tabellenkalkulationsprogrammen üblichen Form bezeichnet: Tabellenname!SpalteZeile, wobei Spalte ein oder mehrere Buchstaben enthalten kann (A, B, ..., Z, AA, AB, ...), Zeile eine 1-basierende Zeilennummer ist und Tabellenname optional in einfachen Anführungszeichen stehen kann. Als Tabellenname kann optional auch die (1-basierende) Nummer der Tabelle in der Arbeitsmappe angegeben werden. Ist kein Tabellenname angegeben, so wird die erste Tabelle der Arbeitsmappe verwendet.

Diese Funktion kann auch über Kommandozeilenparameter verwendet werden.
