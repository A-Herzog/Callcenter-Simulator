Dialog "Skill-Level bearbeiten"

Der Dialog zum Bearbeiten von Skill-Levels wird über die Dialogseite Skill-Level der Agenten des Calcenter-Modell-Editors aufgerufen. In dem Dialog "Skill-Level bearbeiten" können Einstellungen zu einem einzelnen Skill-Level vorgenommen werden.

Jedes Skill-Level stellt einen Satz von Befähigungen zur Bedienung bestimmter Kundentypen dar. Jeder Agentengruppe muss ein Skill-Level zugewiesen werden.

Über die Schaltflächen Skill hinzufügen und Skill entfernen kann angegeben werden, welche Kundentypen Agenten des gewählten Skill-Levels bedinen können soll.

Pro Kundentyp können folgende Einstellungen vorgenommen werden:

  Bedienzeitverteilung

Wird ein Kunde des gewählten Typs von einem Agent dieses Skill-Levels bedient, so wird gemäß der hier angegebenen Bedienzeitverteilung bestimmt, wie lange das Gespräch dauert.

Es kann eine globale Verteilung vorgegeben werden. Außerdem kann pro Intervall vorgegeben werden, dass die globale Verteilung nicht verwendet werden soll (dafür die Auswahlbox "Globale Verteilung für dieses Intervall verwenden" für das jeweilige Intervall deaktivieren) und stattdessen für das jeweilige Intervall eine eigene Verteilung verwendet werden soll.

  Nachbearbeitungszeitverteilung

Wird ein Kunde des gewählten Typs von einem Agent dieses Skill-Levels bedient, so wird gemäß der hier angegebenen Nachbearbeitungszeitverteilung bestimmt, wie lange Nachbearbeitungszeit des Agenten nach dem Gespräch dauern soll, bevor er sich wieder als verfügbar meldet.

Es kann eine globale Verteilung vorgegeben werden. Außerdem kann pro Intervall vorgegeben werden, dass die globale Verteilung nicht verwendet werden soll (dafür die Auswahlbox "Globale Verteilung für dieses Intervall verwenden" für das jeweilige Intervall deaktivieren) und stattdessen für das jeweilige Intervall eine eigene Verteilung verwendet werden soll.

  Score

In jedem Skill-Level kann jedem Kundentyp, den Agenten dieses Skill-Levels bedienen können, eine Scorewert zugewiesen werden. Je höher der Wert ist, desto eher wird ein wartender Kunde des jeweiligen Typs zu einem freien Agenten diesen Skill-Levels geschickt.

Die Skill-Level Score bildet zusammen mit der Callcenter-Score und der Score, die sich aus der Leerlaufzeit des Agenten berechnet, die Agenten-Score.
