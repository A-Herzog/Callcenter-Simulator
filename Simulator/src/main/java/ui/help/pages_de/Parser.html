Grundrechenarten

Unterstützte Anweisungen für die Grundrechenarten: +, -, *, / und ^.

Außerdem steht die If-Funktion zur Verfügung. Diese erwartet eine ungerade Anzahl an Parametern:

If(bedingung1,wert1,bedingung2,wert2,...,wertSonst)

Ist bedingung1>0, so liefert die Funktion wert1 zurück.
Andernfalls prüft sie, ob bedingung2>0 ist und liefert, wenn dies zutrifft, wert2 zurück usw.
Trifft keine der Bedingungen zu, so liefert die Funktion wertSonst zurück.
