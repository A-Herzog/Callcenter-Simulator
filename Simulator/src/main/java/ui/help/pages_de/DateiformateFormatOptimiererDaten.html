Das Basis-Element für Optimierer-Ergebnisse-Dateien heißt <OptimiererDaten>.

Allgemeine Informationen

Jede Optimierer-Ergebnisse-Datei muss von jedem der beiden folgenden Elemente jeweils genau eins enthalten:

  • <Laufzeit>: Enthält die Gesamtlaufzeit der Optimierung in Sekunden.
  • <Anzahl>: Enthält die Gesamtanzahl an Simulationsläufen, die für den Optimierungsprozess durchgeführt wurden.
  • <OptimiererEinstellungen>: Enthält einen Optimierer-Einstellungen-Datensatz, wie auf der Seite Optimierer-Einstellungen beschrieben.

Informationen zu den einzelnen Simulationsläufen

Jede Optimierer-Ergebnisse-Datei enthält ein oder mehrere Elemente vom Typ <Simulationslauf>.

Jedes <Simulationslauf>-Element enthält ein Nummer-Attribut, welches die Nummer der Simulationslaufes innerhalb des Optimierungsprozesses bezeichnet, auf das sich die folgenden Daten beziehen.

Als Inhalt enthält jedes <Simulationslauf>-Element genau ein <Statistik>-Element, wie auf der Seite Statistik-Daten beschrieben.
