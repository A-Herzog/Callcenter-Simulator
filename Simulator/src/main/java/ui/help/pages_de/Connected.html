Simulation > Mehrtägige Simulation

Verkettete Simulation mit Übertrag der Warteabbrecher

Bei der verketteten Simulation werden mehrere Tage mit individuellen Charakteristiken nacheinander simuliert, wobei die endgültigen Warteabbrecher eines Tages zu einem gewissen, konfigurierbaren Anteil als zusätzliche Erstanrufer in den nächsten Tag übernommen werden. Auf diese Weise lässt sich das Fortschreiten einer Überlastwelle über mehrere Tage nachbilden.

Diese Funktion kann auch über Kommandozeilenparameter verwendet werden.

Basisverzeichnis

Hier kann ein beliebiges Verzeichnis angegeben werden. Befinden sich die weiter unten in dem Dialog angegebenen Modell- und Statistikdateien in diesem Verzeichnis, so wird jeweils nur der Dateiname ohne den vollständigen Pfadnamen angezeigt.

Statistik für Übertrag zu Tag 1 (optional)

Bei der mehrtägigen Simulation werden Kundenanfragen, die über 0 Uhr hinaus warten, automatisch inkl. ihrer bisherigen Wartezeit in den nächsten Simulationstag hinein übernommen. Um auch Daten in den ersten Simulationstag der mehrtägigen Simulation hinein übernehmen zu können, kann in diesem Feld eine Statistikdatei angegeben werden, die die Ergebnisse eines dem ersten Tag vorgelagerten Tages enthält.

Zusätzlicher manueller Übertrag in Tag 1 hinein

Zusätzlich oder alternativ zu dem Übertrag in Tag 1 hinein basierend auf der im vorherigen Feld anzugebenden Statistikdatei ist es auch möglich, manuell einen Bestand an wartenden Anfragen, die bereits initial zu Beginn von Tag 1 vorhanden sein sollen, anzugeben.
