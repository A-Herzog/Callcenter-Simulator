Simulationsergebnisse verschiedener Modelle vergleichen

Mit Hilfe dieser Funktion können Sie die Statistik-Ergebnisse von zwei bis vier Simulationsläufen direkt vergleichen. Wenn Sie in der Baumstruktur in der linken Fensterhälfte eine Kategorie auswählen, werden in der rechten Hälfte die zugehörigen Daten der ausgewählten Simulationsläufe neben- oder untereinander angezeigt. Auf diese Weise können mehrere Statistikdatensätze unmittelbar in Bezug auf bestimmte Eigenschaften miteinander verglichen werden.
