Dialog "Schwellenwerte"

In dem Schwellenwerte-Dialog können Werte für die Kenngrößen Wartezeit, Verweilzeit, Erreichbarkeit, Service-Level und Auslastung definiert werden, bei deren Über- oder Unterschreit in den Statistik-Ergebnissen Warnungen angezeigt werden. Es können mehrere Schwellenwerte für verschiedene Kenngrößen definiert werden und pro Kenngröße können zwei Werte, die jeweils zu einer gelben und einer roten Warnung fühen, angegeben werden. Des Weiteren kann angegeben werden, ob die Kenngröße jeweils über alle oder nur über bestimmte Kunden- bzw. Agentengruppen erhoben werden und ob nur einzelne Intervalle oder der gesamte Tag betrachtet werden soll.
