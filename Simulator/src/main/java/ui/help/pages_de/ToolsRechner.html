Rechner

Der Rechner ermöglicht es, schnell mathematische Ausdrücke zu berechnen. Es werden dabei die auf der Seite Rechenbefehle aufgeführten Anweisungen unterstützt. Dieselben Befehle stehen auch zur Verfügung, um z.B. die Anzahl an verfügbaren Warteplätzen in Abhängigkeit von der Anzahl an aktiven Agenten zu steuern (siehe Dialog "Globale Parameter").
